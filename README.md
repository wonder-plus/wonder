
### 1. 問い合わせフォームのメール送信機能（保留中）
現状のフォームは **`mailto:` 方式**（送信ボタンで訪問者のメールアプリが起動し、本人が送信）。
サーバー不要・無料だが、メールアプリ未設定の端末では使えない／ひと手間かかる。

**「フォーム送信＝自動でメール受信」にする場合の選択肢：**

| 方法 | 費用 | 難易度 | 備考 |
|---|---|---|---|
| FormSubmit.co | 無料・登録不要 | かんたん | `form`の`action`をURLに変えるだけ。初回のみ確認メール承認が必要 |

## 開発メモ
- 画像は `photo/` のオリジナルを ffmpeg で長辺1600pxにリサイズ・圧縮して `images/` に配置。
- ファビコン・メニューサムネはヘッドレスChromeでHTML→PNG生成（`docs/menu-thumbs/_build/`）。
- ローカル確認は `index.html` をブラウザで直接開くだけ（サーバー不要）。
- ローカル専用の制作補助物（`docs/menu-plan.html`・`docs/menu-thumbs/`・`docs/stores-text/`）は `.gitignore` 済み。
